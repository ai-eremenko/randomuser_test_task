## Random user app - тестовое задание
Мобильное приложение для Android, предназначенное для получения и отображения данных случайно сгенерированных пользователей.

## Функциональность

- Генерация профиля - получение полного профиля пользователя через интенты пользователя
- Детализированный просмотр - навигация на экран деталей с передачей состояния  
- Интеграция с системными сервисами - открытие карт, инициирование звонков и отправка email через SideEffects

## Технологический стек

- **Язык**: Kotlin
- **Архитектура**: MVI + Clean Architecture
- **UI**: Jetpack Compose
- **DI**: Koin
- **Сеть**: Retrofit 2 + OkHttp3
- **Изображения**: Coil
- **Навигация**: Navigation Component с безопасной передачей аргументов

## Особенности реализации

Приложение реализует паттерн MVI (Model-View-Intent) и соответствует принципам чистой архитектуры. Пользовательский интерфейс построен с использованием Jetpack Compose. Сетевое взаимодействие построено на основе Retrofit 2, а низкоуровневые сетевые операции выполняются с помощью OkHttp3.
